# Contributing to SkillForge AI

Thank you for your interest in contributing to SkillForge AI!

We welcome contributions that improve the project and help build a better learning platform for students.

---

## How to Contribute

1. Fork the repository.

2. Create a new branch.

```
git checkout -b feature/your-feature-name
```

3. Commit your changes.

```
git commit -m "Add: meaningful description"
```

4. Push your branch.

```
git push origin feature/your-feature-name
```

5. Open a Pull Request.

---

## Contribution Guidelines

* Follow clean coding practices.
* Keep components reusable.
* Write meaningful commit messages.
* Test your code before submitting.
* Document new features.

---

## Reporting Bugs

Please include:

* Operating System
* Browser
* Steps to reproduce
* Expected behavior
* Actual behavior
* Screenshots (if applicable)

---

## Suggesting Features

Feature requests are encouraged.

Please describe:

* Problem
* Proposed Solution
* Benefits
* Additional Information

---

## Code Style

* Use descriptive variable names.
* Keep components modular.
* Avoid duplicate code.
* Follow React best practices.

---

Thank you for helping improve SkillForge AI!
