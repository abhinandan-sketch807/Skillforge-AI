# 🚀 SkillForge AI

> **An AI-Powered Full Stack Student Skill Development Platform**

SkillForge AI is a modern full-stack web application designed to help students identify, develop, and showcase their skills through AI-driven recommendations, personalized learning paths, analytics, and gamification.

---

## 📖 Overview

SkillForge AI aims to bridge the gap between academic education and industry requirements by providing an intelligent platform that guides students throughout their learning journey.

The platform combines Artificial Intelligence, Data Analytics, and Gamification to create a personalized learning experience while helping students track progress, improve employability, and prepare for their careers.

# 🚀 Installation

Clone the repository

```
git clone https://github.com/abhinandan-sketch807/Skillforge-AI.git
```

Move into the project

```
cd Skillforge-AI
```

Install frontend dependencies

```
cd frontend
npm install
```

Start frontend

```
npm run dev
```

Backend

```
cd backend
npm install
npm run dev
```
